Nominatim AC

A placename autocomplete service based on OpenStreetMap Nominatim data (source code).

Usage example (using jQuery autocomplete):

Location name: Profile:




Endpoints

Search
http://nominatim-ac.eelpieconsulting.co.uk/search
Parameters
q(required). The partial place name to match.
profile

Limit results to a particular filtered subset of results.

ie. countryCityTownSuburb
tag

Limit results to a particular OSM tag.

ie. railway|station
lat, lon, radiusLimit results to a given geocircle
country

Limit results to a particular country code

Results trail OpenStreetMap by approximately 10 minutes.
